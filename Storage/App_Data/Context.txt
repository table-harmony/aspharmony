AspHarmony is a digital library management platform where users can:

CORE FEATURES:
- Browse, create, and manage digital books
- Join and create libraries
- Share books with other users
- Use different book storage servers
- Switch between multiple visual themes
- Chat with other users in real-time
- Get AI assistance

USER ROLES:
1. Guests can:
   - View available books and libraries
   - Register or login
   - Chat with AI assistant

2. Registered Users can:
   - Manage their profile
   - Create and manage books
   - Join libraries
   - Borrow books
   - Provide feedback
   - Use live chat

3. Administrators can:
   - Manage all libraries
   - Manage user accounts
   - Access admin dashboard

BOOK SERVERS:
A server is a way to store the book information.

Aether: A SOAP web service that stores book data in XML format for external access.
Atlas: An API-based server that stores book data in JSON files locally.
Dummy: A mock server that generates dummy book data for testing purposes.
Echo: Direct-access server using an in-memory DataSet to store book data in XML format.
Harmony: A server that integrates with a Minecraft plugin to store books as in-game items via a Java-based API.
Nimbus 1.0: A SQL Server-based solution storing books in two relational tables: chapters and metadata.
Nimbus 2.0: An updated version of Nimbus, storing book data in a NoSQL MongoDB database.
Orion: A SOAP web service storing book data in XML files via a WebServices project.
Solace: Stores book data in Microsoft Office files (Word, Excel, or PowerPoint) selected by the developer.
Stegan 1.0: Encrypts book data into random images and stores them locally in the project.
Stegan 2.0: Encrypts book data into random images, uploads them online, and stores the links in a database.

HOW TO:
1. Find Books:
   - Use search bar on Books page
   - Filter by server type
   - Browse library collections

2. Join Libraries:
   - Browse available libraries
   - Click "Join Library" button
   - Wait for approval if required

3. Create Books:
   - Click "Create New Book" button
   - Fill required information
   - Choose appropriate server
   - Upload book content

For technical support or questions, use the AI assistant or submit feedback through the Feedback form.
